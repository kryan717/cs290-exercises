Sparc

Sparc is a market place for people to monetize their expertise through in-person and online hosted engagements. It is similar to Uber or Airbnb except the asset sold is the abilities of the person.

Project Road Map

Major Goals

  • Database with information from clients and experts
  • Match undergraduates with graduate students through Airbnb or Tinder like system
  • Coordinate engagenments on Duke's campus through using this web app
  • Create a marketplace for human engagements

First Sprint

  1. Create Wireframes for Login Page
  2. Define schema for database
  3. Work with Laurel on key points of information needed from users
  4. Decide on the frameworks to be used for the app
  5. Begin coding basic frames of the first page
  6. Learn how to code with frameworks chosen
  7. Finish preliminary demo front-end interface/basic website for proposed engagement session
  8. Set up backend database in Flask Python Framework for profile information collection
